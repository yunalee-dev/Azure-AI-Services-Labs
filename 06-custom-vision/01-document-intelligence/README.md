# 01 Document Intelligence

Azure AI Document Intelligence를 사용해 문서에서 텍스트, 표, 키-값 쌍을 추출하는 실습입니다.

## 실습 구성

1. Document Intelligence Studio 실습
2. REST API로 문서 분석
3. Gradio 웹앱으로 문서 업로드 분석

## 사용 기술

- Azure AI Document Intelligence
- Python
- REST API
- Gradio
- Jupyter Notebook

## 폴더 설명

- 01-studio-practice: Studio 기반 클릭 실습
- 02-rest-api: REST API 호출 코드
- 03-gradio-app: Gradio 웹앱 코드
- notebooks: 학습용 노트북
- sample-data: 테스트 문서
- outputs: 분석 결과 저장
