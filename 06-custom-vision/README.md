# 06. Azure Custom Vision

Azure Custom Vision을 사용해 이미지 분류 모델과 객체 감지 모델을 만드는 실습입니다.

## 실습 내용

1. 이미지 분류 모델 만들기
2. 객체 감지 모델 만들기
3. Python SDK로 학습 및 예측 실행
4. Gradio로 간단한 테스트 화면 구성
5. 학습 결과 및 예측 결과 저장

## 폴더 구조

- 01-image-classification: 이미지 분류 실습 코드
- 02-object-detection: 객체 감지 실습 코드
- notebooks: Jupyter Notebook 실습
- sample-data: 학습 및 테스트용 이미지
- outputs: 예측 결과, 모델 내보내기 파일, 실행 화면 캡처

## 사용 기술

- Azure Custom Vision
- Python
- Azure Custom Vision SDK
- Gradio
- Pillow
